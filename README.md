# Farm Stand

Jednoduchá Express + MongoDB aplikace pro správu produktů.

## Spuštění

1. Nainstaluj závislosti:
   ```bash
   npm install
   ```
2. Spusť aplikaci:
   ```bash
   npm start
   ```
3. Otevři v prohlížeči:
   ```text
   http://localhost:3000/products
   ```

## Funkce

- zobrazení všech produktů
- přidání nového produktu
- úprava produktu
- smazání produktu
- jednoduché CSS styly
